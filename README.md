# convert.yazi

## Requirements

- [yazi](https://github.com/sxyazi/yazi)
- [magick](https://archlinux.org/packages/extra/x86_64/thunar/)

## Installation

### Linux 

```sh
git clone https://github.com/atareao/convert.yazi ~/.config/yazi/plugins/convert.yazi
```

or

```sh
ya pack -a atareao/convert
```
